1 Datengrundlage

Wir verwenden in diesem Beispiel Daten des Tennessee Student/Teacher Achievement Ratio (STAR) Projekts, das in den 1980er Jahren durchgeführt wurde. In diesem Projekt wurden Schüler:innen über mehrere Jahre hinweg begleitet, um die Auswirkungen von Klassengröße auf die akademische Leistung zu untersuchen. Die Daten enthalten Informationen über verschiedene Aspekte der Schüler*innen, wie z.B. ihre akademischen Leistungen, Verhaltensweisen und ihren sozioökonomischen Hintergrund.

Für die konfirmatorische Faktorenanalyse verwenden wir die Items des STUDENT PARTICIPATION QUESTIONNAIRE (STPQ), der von Lehrer:innen ausgefüllt wurde, um die Beteiligung der Schüler:innen am Unterricht zu bewerten.

Wir arbeiten heute mit 10 Items, die den Skalen E = Engagement, I = Initiative und N = Nonparticipatory Behavior zugeordnet werden können.

Skala Wortlaut Itemname
E+ pays attention in class g4ptattn
E- loses, forgets, or misplaces materials g4ptmtrl
E- comes late to class g4ptlate
E+ completes assigned seat work g4ptwork
E+ is persistent when confronted with difficult problems g4ptpers
N+ needs to be reprimanded g4ptrepr
N+ annoys or interferes with peers’ work g4ptanoy
I+ participates actively in discussions g4ptdisc
I+ does more than just the assigned work g4ptextr
I+ engages teacher in conversation about subject matter before or after school, or outside of class g4ptdiss

Alle Items wurden auf einer 5-stufigen Skala von 1 = nie bis 5 = immer bewertet.

Warnung

Der Datensatz enthält eine genestete Struktur, da Schüler:innen in Klassen und Schulen organisiert sind. Um nicht gleich mit Multi-Level-Konfirmatorischen-Faktorenanalysen starten zu müssen, habe ich jeweils eine Schülerin bzw. einen Schüler zufällig aus jeder Klasse ausgewählt. In der Praxis sollte man dies natürlich nicht tun.

3 Messmodellvergleiche

Wir können nun verschiedene Messmodelle spezifizieren und vergleichen, um zu sehen, welches am besten zu den Daten passt. Dazu fangen wir am besten mit der Engagement-Skala an und spezifizieren zunächst ein \(\tau\)-generisches Messmodell. Dazu verwenden wir das {lavaan} Paket und spezifizieren das Modell in der sogenannten lavaan-Syntax, die sich an die natürlichsprachliche Beschreibung von Modellen anlehnt und hier sowie hier übersichtlich dargestellt wird.

3.1 \(\tau\)-generisches Messmodell

library(lavaan)
This is lavaan 0.6-21
lavaan is FREE software! Please report any bugs.
model_tau <- '
E =~ g4ptattn + g4ptmtrl + g4ptlate + g4ptwork + g4ptpers
'

In dieser Syntax bedeutet E =~ g4ptattn + g4ptmtrl + g4ptlate + g4ptwork + g4ptpers, dass die latente Variable E durch die fünf Items g4ptattn, g4ptmtrl, g4ptlate, g4ptwork und g4ptpers gemessen wird. Das Symbol =~ wird verwendet, um anzugeben, dass es sich um eine Messmodell-Spezifikation handelt, während ~ eine Regression und ~~ eine Varianz oder Kovarianz spezifiziert. Die Items auf der rechten Seite des =~-Operators sind die manifesten Indikatoren, die die latente Variable E messen. Ähnlich wie bei der Spezifikation von Regressionen in R bedeutet ~ + g4ptmtrl, dass eine multiplikative Konstante (Ladung, Regressionsgewicht) frei für g4ptmtrl geschätzt wird. Das Objekt model_tau enthält aber nur die Spezifikation unseres \(\tau\)-generischen Modells, dessen Parameter wir nun anhand der Daten schätzen:

fit_tau <- cfa(model_tau, data = data_star)

3.2 \(\tau\)-paralleles Messmodell

Das \(\tau\)-generische Messmodell oben zeigte sehr hohe Fit-Indices, aber es ist natürlich auch sehr flexibel, da alle Ladungen frei geschätzt werden können. Ein \(\tau\)-paralleles Messmodell schränkt alle Ladungen auf einen Wert ein, was annimmt, dass alle Items gleich trennscharf sind.

model_tau_parallel <- '
E =~ a*g4ptattn + a*g4ptmtrl + a*g4ptlate + a*g4ptwork + a*g4ptpers
'
fit_tau_parallel <- cfa(model_tau_parallel, data = data_star)

3.3 Latente (messfehlerbereinigte) Korrelationen

In diesem Abschnitt wollen wir die latente Korrelation zwischen den Skalen E, I und N schätzen und mit der manifesten messfehlerbehafteten Korrelation vergleichen. Dazu müssen wir zunächst die invers kodierten Items rekodieren und die drei Skalen spezifizieren:

data_star <- data_star |>
  mutate(
    E_scalescore = rowMeans(
      pick(g4ptattn, g4ptmtrl_rec, g4ptlate_rec, g4ptwork, g4ptpers),
      na.rm = T
    ),
    I_scalescore = rowMeans(pick(g4ptdisc, g4ptextr, g4ptdiss), na.rm = T),
    N_scalescore = rowMeans(pick(g4ptrepr, g4ptanoy), na.rm = T)
  )
mod_participation <- '
E =~ g4ptattn + g4ptmtrl_rec + g4ptlate_rec + g4ptwork + g4ptpers
I =~ g4ptrepr + g4ptanoy
N =~ g4ptdisc + g4ptextr + g4ptdiss
'

fit_participation <- cfa(mod_participation, data = data_star)
summary(fit_participation, fit.measures = TRUE, standardized = TRUE)

3.3.3 Minderungskorrektur (Attenuation Correction)

Die manifesten Korrelationen sind durch Messfehler attenuiert. Mit Spearman’s Minderungskorrektur können wir die wahre Korrelation schätzen:

\[ r_{xy,\text{korrigiert}} = \frac{r_{xy}}{\sqrt{\text{Rel}_x \cdot \text{Rel}_y}} \]

# Reliabilitäten (Cronbach's Alpha) berechnen
library(psych)

    %+%, alpha
The following object is masked from 'package:lavaan':

    cor2cov
alpha_E <- alpha(data_star |> select(g4ptattn, g4ptmtrl_rec, g4ptlate_rec, g4ptwork, g4ptpers))$total$raw_alpha
alpha_I <- alpha(data_star |> select(g4ptrepr, g4ptanoy))$total$raw_alpha
alpha_N <- alpha(data_star |> select(g4ptdisc, g4ptextr, g4ptdiss))$total$raw_alpha

# Minderungskorrektur anwenden
cor_EI_corrected <- cor_manifest["E_scalescore", "I_scalescore"] / sqrt(alpha_E * alpha_I)
cor_EN_corrected <- cor_manifest["E_scalescore", "N_scalescore"] / sqrt(alpha_E * alpha_N)
cor_IN_corrected <- cor_manifest["I_scalescore", "N_scalescore"] / sqrt(alpha_I * alpha_N)

# Vergleichstabelle erstellen
tibble(
  Korrelation = c("E - I", "E - N", "I - N"),
  Manifest = c(
    cor_manifest["E_scalescore", "I_scalescore"],
    cor_manifest["E_scalescore", "N_scalescore"],
    cor_manifest["I_scalescore", "N_scalescore"]
  ),
  `Minderungskorrigiert` = c(cor_EI_corrected, cor_EN_corrected, cor_IN_corrected),
  Latent = parameterEstimates(fit_participation, standardized = TRUE) |>
    filter(op == "~~" & lhs != rhs & lhs %in% c("E", "I", "N")) |>
    pull(std.all)
)

Die latenten Korrelationen aus der CFA und die minderungskorrigierten Korrelationen sollten ähnliche Werte aufweisen, da beide Ansätze die Korrelation um Messfehler bereinigen. Die manifesten Korrelationen sind systematisch niedriger, da sie durch die Unreliabilität der Messungen attenuiert sind.
